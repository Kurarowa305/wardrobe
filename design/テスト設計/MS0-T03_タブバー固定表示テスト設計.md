# MS0-T03 タブバー固定表示テスト設計

## 1. 目的

* タブバーが端末画面下部に固定表示され、タブ付き画面の本文と重ならないことを静的検証で担保する。
* セーフエリアを考慮した下部余白が設定されていることを確認し、モバイル端末で操作不能にならないようにする。

## 2. 対象

* `apps/web/src/app/globals.css`
* `apps/web/scripts/check-tab-bar-fixed-spec.mjs`
* `.github/workflows/ci.yml`

## 3. テスト観点

### 3.1 表示位置

* `tab-bar` に `position: fixed` が設定されていること
* `bottom: 0` で画面最下部に固定されること
* `left: 50%` と `transform: translateX(-50%)` により中央配置されること
* `width: min(100%, 420px)` によりアプリの最大幅に追従すること
* `z-index: 20` により本文より前面に表示されること

### 3.2 モバイル端末考慮

* `padding-bottom: env(safe-area-inset-bottom)` によりセーフエリアへ追従すること

### 3.3 本文との重なり防止

* `app-content` に `calc(16px + 56px + env(safe-area-inset-bottom))` の下余白があり、固定タブバーと重ならないこと

### 3.4 CI組み込み

* `pnpm --filter web test:tab-bar-fixed` が `apps/web/package.json` に定義されていること
* GitHub Actions の CI で当該テストが実行されること

## 4. テストケース

| ケースID | 観点 | 条件 | 期待結果 |
| --- | --- | --- | --- |
| TBF-01 | 表示位置 | `globals.css` を確認する | `tab-bar` に fixed 下部配置・中央寄せ・最大幅・前面表示の指定がある |
| TBF-02 | 重なり防止 | `globals.css` を確認する | `app-content` に固定タブバー高さとセーフエリアを含む下余白がある |
| TBF-03 | CI組み込み | `package.json` と `ci.yml` を確認する | 固定表示テストの npm script が定義され、CI から実行される |

## 5. 実装方針

* Node.js スクリプト `apps/web/scripts/check-tab-bar-fixed-spec.mjs` で必要文字列の有無を検証する。
* PR・push 時の CI で継続的に実行し、退行を防止する。
