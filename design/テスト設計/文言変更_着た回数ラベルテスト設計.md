# 文言変更（着た回数ラベル）テスト設計

## 目的

- 服詳細・テンプレート詳細・履歴詳細の表示文言が、要望どおり自然な表現へ統一されていることをCIで継続検証する。
- 文言設計の集約先である `features/*/strings.ts` に変更を閉じ込め、画面側の再利用文言が意図せず旧表現へ戻らないことを担保する。

## 対象スクリプト

- `apps/web/scripts/check-copy-labels-spec.mjs`
- 実行コマンド: `pnpm --filter web test:copy-labels`

## テスト観点

### CPL-01 服詳細文言が更新されている
- 観点: 服詳細画面の利用状況ラベルが自然な日本語へ変更されていること
- 期待結果:
  - `src/features/clothing/strings.ts` に `wearCount: "着た回数"` が定義される
  - `src/features/clothing/strings.ts` に `lastWornAt: "最後に着た日"` が定義される

### CPL-02 テンプレート詳細文言が更新されている
- 観点: テンプレート本体と構成服の両方で、旧表現が残らないこと
- 期待結果:
  - `src/features/template/strings.ts` に `wearCount: "着た回数"` が定義される
  - `src/features/template/strings.ts` に `lastWornAt: "最後に着た日"` が定義される
  - `src/features/template/strings.ts` に `clothingWearCount: "服の着た回数"` が定義される
  - `src/features/template/strings.ts` に `clothingLastWornAt: "服の最後に着た日"` が定義される

### CPL-03 履歴詳細文言が更新されている
- 観点: 履歴詳細のテンプレートカード/服カード見出しが要望どおり変更されていること
- 期待結果:
  - `src/features/history/strings.ts` に `template: "着たテンプレート"` が定義される
  - `src/features/history/strings.ts` に `clothingItems: "着た服"` が定義される
  - `src/features/history/strings.ts` に `templateWearCount: "着た回数"` が定義される
  - `src/features/history/strings.ts` に `templateLastWornAt: "最後に着た日"` が定義される
  - `src/features/history/strings.ts` に `clothingWearCount: "着た回数"` が定義される
  - `src/features/history/strings.ts` に `clothingLastWornAt: "最後に着た日"` が定義される

## CI適用方針

- `.github/workflows/ci.yml` に `Copy labels spec test` を追加し、push ごとに `pnpm --filter web test:copy-labels` を実行する。

## PRサマリーに記載するテストケース

- CPL-01 服詳細文言が「着た回数」「最後に着た日」へ更新されている
- CPL-02 テンプレート詳細文言が「着た回数」「最後に着た日」へ更新され、構成服文言も追随している
- CPL-03 履歴詳細文言が「着たテンプレート」「着た服」「着た回数」「最後に着た日」へ更新されている
