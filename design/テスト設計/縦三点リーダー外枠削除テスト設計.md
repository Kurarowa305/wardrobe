# 縦三点リーダー外枠削除テスト設計

## 1. 目的
- 詳細画面ヘッダー右上の縦三点リーダートリガーから円形の外枠が削除されていることを担保する。
- 外枠削除の検証スクリプトが package.json と CI に登録され、継続的に検知できることを担保する。

## 2. 対象
- `apps/web/src/app/globals.css`
- `apps/web/scripts/check-overflow-menu-trigger-spec.mjs`
- `apps/web/package.json`
- `.github/workflows/ci.yml`

## 3. テスト観点
1. `.overflow-menu-trigger` に対して border が削除されている。
2. `.overflow-menu-trigger` に対して border-radius が無効化され、背景が透明化されている。
3. 検証スクリプト実行コマンドが `apps/web/package.json` と `.github/workflows/ci.yml` に追加されている。

## 4. テストケース
| ID | 観点 | 確認内容 | 期待結果 |
| --- | --- | --- | --- |
| OMT-01 | 外枠削除 | `globals.css` の `.overflow-menu-trigger` 定義を確認する | `border: 0;` `border-radius: 0;` `background: transparent;` が設定されている |
| OMT-02 | 継続検知 | package.json と CI workflow に新テストが登録されているか確認する | ローカル・CI の両方で `test:overflow-menu-trigger` を実行できる |

## 5. 実行方法
- `pnpm --filter web test:overflow-menu-trigger`
- 必要に応じて関連既存テストとして `pnpm --filter web test:overflow-menu` を併せて実行する。
