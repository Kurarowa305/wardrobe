# 履歴日付曜日表示テスト設計

## 目的
- 履歴カード（ホーム・履歴一覧）と履歴詳細画面の日付が曜日付き `YYYY/MM/DD (曜)` 形式で表示されることを静的検証する。
- 最終着用日を表示する全画面（服詳細・テンプレート詳細・履歴詳細の着用服）で曜日付き表示へ統一されることを確認する。
- 履歴詳細画面のテンプレート情報から「最後に着た日」が削除されていることを確認する。

## 対象
- `apps/web/src/features/history/date.ts`
- `apps/web/src/components/app/history/HistoryCard.tsx`
- `apps/web/src/components/app/screens/HistoryDetailScreen.tsx`
- `apps/web/src/components/app/screens/ClothingDetailScreen.tsx`
- `apps/web/src/components/app/screens/TemplateDetailScreen.tsx`
- `apps/web/scripts/check-history-date-weekday-spec.mjs`
- `.github/workflows/ci.yml`

## テスト観点
1. `formatHistoryDate` が `YYYY/MM/DD (曜)` 形式を返す。
2. `formatLastWornDate` が最終着用日を曜日付きで整形し、未着用時は既存ラベルを返す。
3. ホーム・履歴一覧の履歴カードが共通フォーマッタで曜日付き日付を表示する。
4. 履歴詳細画面が曜日付き日付を表示し、テンプレート・着用服の「最後に着た日」表示を持たない。
5. 服詳細・テンプレート詳細・テンプレート配下の服一覧が曜日付き最終着用日を表示する。
6. 上記静的検証が CI で実行される。

## テストケース
- **HDW-01**: 履歴日付フォーマッタが曜日配列を持ち、`YYYY/MM/DD (曜)` を返す。
- **HDW-02**: 最終着用日フォーマッタが未着用ラベルと曜日付き日付の両方を扱う。
- **HDW-03**: 履歴カードが `formatHistoryDate` を使って日付を表示する。
- **HDW-04**: 履歴詳細画面が曜日付き日付を表示し、テンプレート・着用服の最後に着た日行を含まない。
- **HDW-05**: 服詳細・テンプレート詳細・テンプレート配下の服一覧が `formatLastWornDate` を使う。

## 実行コマンド
- `pnpm --filter web test:history-date-weekday`
